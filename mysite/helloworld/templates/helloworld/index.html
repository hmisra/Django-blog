Projects

Othello game playing agent using Mini-Max, Alpha Beta Pruning algorithm and a custom Heuristic Function
A game playing agent which uses Mini-Max and Alpha Beta Pruning algorithm to calculate the best possible move using a custom heuristic function which uses the weighted board position to calculate the heuristic value to determine how favorable the given game board is and combine that information with Game Data collected from previous world championships to determine its next move. The agent was able to go till depth 12 in the game tree using alpha beta pruning in 6 seconds to calculate the best possible move.
