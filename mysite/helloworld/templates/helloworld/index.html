Projects

Othello game playing agent using Mini-Max, Alpha Beta Pruning algorithm and a custom Heuristic Function
Designed and developed an Othello game playing agent using Mini-Max and Alpha Beta Pruning algorithm. The agent used a heuristic function which uses the weighted board positions to calculate the heuristic value to determine how favorable the given game board is and combine that information with Game Data collected from previous world championships to determine its next move. The agent was able to go till depth 12 in the game tree in 6 seconds using alpha beta pruning to calculate the best possible move.
